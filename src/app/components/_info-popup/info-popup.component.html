This text is intended to help you understand how this app works.
In a nutshell, the application works according to the following scheme:

1. We start by creating a goal, e.g. 'I will run a marathon'. We do this by clicking on the sidemenu tab 'My goals' -> by clicking on 'new goal', a modal window will open, in which we create a new goal, where are various options to choose from, such as the field of life (career, knowledge, health), deadline, etc.

2. After creating the goal (point 1) click on the 'Calendar' tab -> here by clicking 'new task' for the created goal (point 1) we create individual tasks (tasks) to be performed.

Other tools (charts) are used to measure progress and proportions between goals from different ones areas of life so as to help create a balance between individual spheres of life. In the 'settings' tab you can edit/change your profile details.

In principle, the operation is similar to a work schedule, except that goals are not set within the work, only goals are about your life.

Enjoy the application.
